アカウント削除リクエスト

LokaShare では、ユーザー自身がいつでもアカウントを削除できます。 本ページでは、アプリ内での削除手順と、Web 経由での削除リクエスト窓口についてご案内します。

方法 1: アプリ内から削除する (推奨・即時反映)

  1. LokaShare アプリを起動してサインイン
  2. 下部メニューの「設定」をタップ
  3. 「アカウント」セクションの「アカウントを削除」をタップ
  4. 確認ダイアログで「削除」をタップ

アプリ内から削除した場合、関連データはすべて即時にサーバーから削除されます (Supabase Auth の auth.users レコード削除に伴う CASCADE 削除)。

方法 2: Web 経由で削除リクエストを送る

アプリにアクセスできない場合や、退会済みでログイン不可の場合は、 下記メールアドレスまでアカウント削除リクエストをお送りください。

送信先: tokiyoshi.0410@gmail.com

件名: 「[LokaShare] アカウント削除リクエスト」

本文に必要な情報:

  • アカウントに登録しているメールアドレス
  • アプリ内のユーザー名 (分かる範囲で)
  • 削除を希望する旨

リクエスト受領後、本人確認のうえ 30 日以内 に削除を実行します。 本人確認のため、登録メールアドレスから送信していただく必要があります。

削除されるデータ

データ種別 削除タイミング
アカウント情報 (メール / ユーザー名 / プロフィール写真) 即時
位置情報履歴 即時
友達関係 / ブロック / フォロー 即時
送受信メッセージ・写真・ストーリー 即時
集合ピン・メモリー・経路記録 即時
緊急連絡先・通知設定 即時
サーバーバックアップ 最大 30 日以内

保持されるデータ (法令遵守目的)

法令により記録保持が義務付けられている場合、以下のデータは匿名化のうえ最大 1 年間保持される場合があります:

  • 不正利用 / 通報対応に関する操作ログ (個人を特定できない形式)
  • 支払い履歴 (該当機能を提供している場合のみ・現状なし)

削除後の挙動

  • 同じメールアドレスで再度新規登録できます。ただし過去のデータは復元されません。
  • 友達リストから自動的に外れます (相手側にも反映)。
  • 送受信したメッセージは、相手側の画面からも削除されます。
